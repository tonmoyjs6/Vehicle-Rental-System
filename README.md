
# Vehicle Rental System.

## Description
It is mainly backend vehicle management sysetem. where a customer can rent vehicle. It allows a user booking vehicle as well as cancel.
Admin can cancel rent of user service as manage user activity.

## Features
Create a user with jwt authentication.
cancel bookings before start date
Admin can marked bookings returned
Auto updtae vehicle availabilty after rent end date.
Admin can delete a user.
Admin can show all users.
Postgress database intregretation.
Jwt based authentication

## Tech Stack
Node Js
Express Js
TypesCript
Jwt Authentication
